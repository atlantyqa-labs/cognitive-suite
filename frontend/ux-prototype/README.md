# UX Prototype (React + Vite)

Prototipo visual para iterar interfaz y experiencia de usuario.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Calidad y SCA

- Linting y formato con `Biome` (`npm run lint`, `npm run lint:fix`).
- Dependencias auditadas con `npm audit`.
