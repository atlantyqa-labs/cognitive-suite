# Self-hosted Fonts

These fonts are vendored for local rendering and to avoid runtime third-party calls.

## Families

- Inter: weights 300, 400, 500, 600, 700
- Space Grotesk: weights 500, 600, 700
- Roboto Mono: weights 400, 700

## Source

Downloaded on 2026-02-16 from Google Fonts static assets (`fonts.gstatic.com`) using pinned URLs.

## Integrity

Checksums are recorded in `SHA256SUMS.txt`.
